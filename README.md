**Simple Node System (DAG) in tkinter python!**

## Feature
- Lightweight library
- Easy to install and use
- Multiple nodes and inputs
- **Save/Load** node trees
- Canvas with **drag and zoom** ability
- Customizable options
- Built-in **Node menu**
